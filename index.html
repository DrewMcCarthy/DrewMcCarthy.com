About Me

Hello, welcome to my webdev sandbox & website for prospective employers. I've been working in the healthcare industry since 2010 as a DBA with a strong focus on database development and supporting our software engineers with data modeling, query tuning, and writing stored code for Oracle and SQL Server.

My passion for the last few years has been learning full-stack web development which for me means a lot of front-end focus in my spare time. I'm comfortable using either Python or .Net on the back-end and I try to use vanilla JS anywhere I can on the front-end. Although more recently I started working with ReactJS and have really enjoyed the development experience especially when used with the static site generator GatsbyJS!

I'm very excited about the progress of web development! It seems like a lot of the tech is maturing now and I appreciate that the major libraries/frameworks put a lot of emphasis on the developer workflow. I especially like the bundling and pre-processing that is common now. I'll be continuing my self-education and updating this and other websites as I learn more. My goal is to focus more on each individual project and add enough polish to make it "production quality" and share my work on GitHub.
