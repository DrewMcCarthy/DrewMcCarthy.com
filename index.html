About Me

Hey there! My name is Drew McCarthy, I've been working as a Database Administrator/Developer in the healthcare industry since 2010. My primary focus is on database development and working with talented software engineers doing data modeling, query tuning, and writing stored code for Oracle and SQL Server. I've also developed some desktop apps for use by clinical and operations staff and have been pivoting toward software development for some time.

My hobby and passion for the last year or two has been learning full-stack web development. Because I have database and desktop development experience I focus on learning front-end concepts and technologies in my spare time. Recently I started working with Javascript frameworks, namely ReactJS and have really enjoyed the development experience. I've also played with the static site generator GatsbyJS after my dissatisfaction with Wordpress during a freelance job and am looking forward to using it in a project!

I'm very excited to be transitioning to web development at this point in it's maturity! Many popular frameworks and libraries are mature enough to learn quickly and have excellent documentation but are not so rigid that they can't be integrated into existing applications. I'll be continuing my self-education and updating this and other websites as I learn more. I'll also be sharing any project that I feel is "production quality" code to my GitHub account. Thanks for visiting!
